//Auto indentar o código ao salvar o arquivo (Não funciona com HTML)
ctrl + shift + p -> digite install -> escolha Package Control: Install Package
ctrl + shift + p -> digite codeformatter -> escolha CodeFormatter
ctrl + shift + p -> digite install -> escolha Package Control: Install Package
ctrl + shift + p -> digite phpfmt -> escolha phpfmt
Configurar o CodeFormatter 
-> Menu Preferences -> Package Settings -> CodeFormatter -> Settings - Default ->
-> copiar todo o conteúdo e cole em -> Menu Preferences -> Package Settings -> CodeFormatter -> Settings - User ->
-> basta deletar os códigos que não serão utilizados 
-> altere a linha "format_on_save": false -> "format_on_save": true
-> para a formatação do php, exclua as seguintes linhas:
"php55_compat": false,
"psr1": false,
"psr1_naming": false,
"psr2": true,
-> na linha "passes": [], poderá ser incluída várias configurações para formatar o código php no link abaixo:
https://github.com/akalongman/sublimetext-codeformatter/blob/master/PHP-Transformations.md
"passes": ["AutoSemicolon", "OrderAndRemoveUseClauses", "OrganizeClass", "ShortArray"],

Obs: caso dê algum erro na hora de salvar o arquivo php, basta incluir 
o caminho do servidor php na linha: "php_path": "C://xampp//php//php.exe",

//Importando classes com namespaces
ctrl + shift + p -> digite install -> escolha Package Control: Install Package
ctrl + shift + p -> digite phpcompanion -> escolha PHP Companion
Acesse o link -> https://github.com/erichard/SublimePHPCompanion/blob/master/Default.sublime-keymap-sample -> copie as configurações as seguintes confugurações ->
{ "keys": ["f6"], "command": "expand_fqcn" },
{ "keys": ["shift+f6"], "command": "expand_fqcn", "args": {"leading_separator": true} },
{ "keys": ["f5"], "command": "find_use" },
{ "keys": ["f4"], "command": "import_namespace" },
{ "keys": ["f3"], "command": "implement" },
{ "keys": ["shift+f12"], "command": "goto_definition_scope" },
{ "keys": ["f7"], "command": "insert_php_constructor_property" }
-> e cole em -> Menu Preferences -> Key Bindings -> na aba Default.sublime-keymap - User
-> altere a linha { "keys": ["f5"], "command": "find_use" }, para -> { "keys": ["ctrl+i"], "command": "find_use" },
Para usar, coloque o ponteiro do mouse em cima da classe e aperte ctrl+i que automaticamente será criado o namespace da classe

//Criando snippets
ctrl + shift + p -> digite install -> escolha Package Control: Install Package
ctrl + shift + p -> digite snippetMaker -> escolha SnippetMaker
ctrl + shift + p -> digite snippet maker ou make -> escolha SnippetMaker: Make Snippet -> 
-> irá aparecer uma caixa com nome Trigger -> para criar o atalho do snippet -> Ex: pub (para criar uma public function)
-> depois irá aparecer uma caixa com o nome Description -> criar descrição do snippet
-> depois irá aparecer uma caixa com o nome Scope -> pode deixar padrão
-> depois irá aparecer uma caixa com o nome File Name -> pode deixar padrão
-> depois irá aparecer uma aba -> abaixo de <content><![CDATA[ deve ser criado o snippet
->Ex: 
public function ${1}(){
    ${2}
}
O ${1} significa a parada do cursor após a criação do snippet
Para criar o snippet -> digite o que foi colocado em Trigger e aperte TAB

//Sql no sublime
ctrl + shift + p -> digite install -> escolha Package Control: Install Package
ctrl + shift + p -> digite sqlto -> escolha SQLTools
É preciso criar um arquivo com a extensão .sql dentro do seu projeto (Ex: sql.sql)
Menu Preferences -> Package Settings -> SQLTools -> Settings e na aba SQLTools.sublime-settings - User cole os códigos abaixo:
"show_result_on_window": true,  (para aparecer o resultado em uma janela/aba separado)
ctrl + shift + p -> digite connection -> escolha ST: Setup Connections -> na aba SQLToolsConnections.sublime-settings - User -> 
-> comente todos as configurações, com exceção do MySQL -> altera as seguintes linhas:
"database": "nome do seu banco de dados",
"username": "nome de usuário",
"password": "senha",
ctrl + shift + p -> digite connection -> escolha ST: Select Connection -> escolha Connection  MySQL (Seleciona a conexão com o DB)
ctrl + shift + p -> digite exe -> escolha ST: Execute (Executa os comandos SQL)
ctrl + shift + p -> digite sql -> escolha Set Syntax: SQL (formatção SQL)
